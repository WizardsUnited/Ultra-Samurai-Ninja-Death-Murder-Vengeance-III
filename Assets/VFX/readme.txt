VFX Folder Structure

The VFX folder is dedicated to visual effects, organizing particles, and related assets.

STORE SHADERS IN SHADERS

Folder Descriptions

    /Particles: Contains particle systems used for effects like fire, smoke, or magic.
    /Textures: Includes textures used by particle systems or other VFX elements.
    /Prefabs: Stores prefabs for complex visual effects that combine multiple elements, like explosions or weather effects.

Examples

    /Particles: A fire particle system might include multiple emitters and shaders to create a realistic flame effect.
