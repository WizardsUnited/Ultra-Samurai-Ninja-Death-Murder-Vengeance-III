Tests Folder Structure

The Tests folder organizes all test scripts and assets related to automated testing in the project.
Folder Descriptions

    /UnitTests: Contains scripts that test individual components or classes, ensuring they function as expected in isolation.
    /IntegrationTests: Includes tests that verify the correct interaction between different components or systems within the game.
    /PlayModeTests: Stores tests that run in Play Mode, simulating in-game conditions to ensure gameplay mechanics function correctly.
    /EditModeTests: Contains tests that run in Edit Mode, verifying editor scripts and tools perform as intended.
