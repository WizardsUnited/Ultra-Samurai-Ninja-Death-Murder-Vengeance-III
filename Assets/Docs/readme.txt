Docs Folder Structure

The Docs folder contains all documentation related to the project, including design documents, technical specifications, and guidelines.
Folder Descriptions

    /DesignDocs: Houses design documents that outline the game's narrative, mechanics, and visual style.
    /TechnicalSpecs: Contains technical specifications and requirements for the project, including platform support and performance targets.
    /API: Stores documentation related to the game's codebase, including class references and API guidelines.
    /Readme: Includes the main README file that provides an overview of the project and essential information for developers.
    /Guidelines: Contains best practices and coding standards to ensure consistency across the project's codebase.
