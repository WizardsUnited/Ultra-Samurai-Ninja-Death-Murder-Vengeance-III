Audio Folder Structure

This Audio folder organizes all sound-related assets, including music, sound effects, and voiceovers.
Folder Descriptions

    /Music: Contains background music tracks used throughout the game.
    /SFX: Stores sound effects, categorized further by their use (e.g., UI, environment, characters).
    /Voice: Contains voiceover files, organized by character or narrative function.

Examples

    /SFX: Inside this folder, you may find subfolders like UI for button clicks and Environment for ambient sounds like wind or water.
