Animation Folder Structure

The Animation folder contains all assets related to animation, including clips, controllers, and rig data.
Folder Descriptions

    /Characters: Stores animation clips and Animator controllers specific to characters.
    /Environment: Includes animations for environmental objects, such as doors opening or trees swaying.
    /Props: Contains animations for props, which might include mechanical movements or item interactions.
    /UI: Houses animations related to UI elements, such as button presses or screen transitions.
