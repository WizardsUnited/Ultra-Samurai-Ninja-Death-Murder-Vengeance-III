Materials Folder Structure

The Materials folder organizes all material assets used to define the appearance of 3D objects in the game.
Folder Descriptions

    /Characters: Contains materials applied to character models.
    /Environment: Houses materials used on environmental assets like terrain, buildings, and vegetation.
    /Props: Includes materials for various props and interactable objects within the game.
    /UI: Stores materials used specifically in the user interface, such as custom button appearances.

Examples

    /Environment: You might find materials here that give different surfaces their specific textures and reflections, like a water material with reflection and ripple effects.
