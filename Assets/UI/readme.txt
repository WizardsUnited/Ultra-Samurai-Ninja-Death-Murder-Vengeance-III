UI Folder Structure

The UI folder organizes all assets related to the game's user interface.
Folder Descriptions

    /Canvases: Contains prefabs and setup for UI canvases that represent different screens or overlays in the game.
    /Elements: Stores reusable UI elements like buttons, sliders, and panels.
    /Fonts: Contains font assets used throughout the UI.
    /Layouts: Houses layout groups and settings for arranging UI elements.
    /Animations: Includes animation assets and controllers specific to UI elements, such as transitions or interactive effects.

Examples

    /Layouts: A layout prefab might organize a dynamic inventory grid, automatically adjusting the positions of items as they are added or removed.
