Scenes Folder Structure

The Scenes folder contains all Unity scene files, representing different levels or parts of the game.
Folder Descriptions

    /MainMenu: Contains the scene(s) for the game's main menu.
    /Levels: Houses individual levels or areas within game.
    /TestScenes: Includes scenes used for testing gameplay features, mechanics, or assets.
    /Cutscenes: Stores scenes dedicated to in-game cinematic sequences.

Examples

    /Levels: Each level of the game is stored as a separate scene file, allowing for easy editing and management of individual stages.
