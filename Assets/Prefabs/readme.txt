Prefabs Folder Structure

The Prefabs folder contains all prefabricated game objects that are reused throughout the game.
Folder Descriptions

    /Characters: Contains prefabbed character models, ready for instantiation in the game.
    /Environment: Houses environmental prefabs like trees, rocks, and buildings.
    /Props: Includes smaller, often interactive, prefabs like doors, levers, and pickups.
    /UI: Prefabs related to user interface elements such as menus, buttons, and HUD components.
    /VFX: Stores prefabs for visual effects, such as explosions or particle systems.
    /Gameplay: Holds prefabs specifically tied to gameplay mechanics, like traps or collectible items.

Examples

    /Characters: A prefab here might include a fully rigged and animated character, complete with collider and script components for behavior.
